This is the official project for the Kippie Godot lectures. Given by Rian Young (me!) hosted in the Kippie offices.

In this project, I taught a group of young aspiring game devs the ropes of the Godot game engine, using version 4.5.1.

# Week 1 - 20/11/25:
- Downloading Godot
- Creating a project
- Navigating the UI
- Short explainer on GDScript and creating objects
- Understanding the hierarchical scene tree
- Quick note on the file system
- Creating a player with a collider and graphics
- Basic built-in character controller
- Ended with showing "Rails of Ruin"

# Week 2 - 27/11/25:
- Showing some of the inner workings of "Rails of Ruin" in-engine
- Creating a platform for our player to stand on
- Some discussion on enemy FSMs and object interaction
- Attaching a camera to the player
- Fixing some issues with node ancestry
- Adding a background to the game

# Week 3 - 04/12/25:
- Refresher on GDScript syntax
- Quick note on the Godot lifecycle
- Adding a generic health component to our game
- Keywords and types in Godot
- Variables in Godot
- Colours of text in the IDE
- Declaring our own functions
- Parameters in functions
- Attaching our component to the player
- Connecting the component signal to a function in the player
- Difference between signals and functions
